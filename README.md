
Aurazone QA Automation
📌 Project Overview
This project contains basic Selenium automation scripts for testing the Aurazone e-commerce website.

🔧 Tools Used
Java
Selenium WebDriver
ChromeDriver
✅ Test Cases Automated
Verify homepage loads successfully
Search for a product
Open product details
Add product to cart
Navigate to cart page
▶️ How to Run
Install Java (JDK 8 or above)
Download ChromeDriver and set path
Import project into Eclipse/IDE
Run AurazoneTest.java
